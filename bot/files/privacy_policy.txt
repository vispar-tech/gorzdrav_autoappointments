<b>Политика конфиденциальности</b>

<i>Дата вступления в силу: 4 сентября 2025 года</i>

Настоящая Политика конфиденциальности (далее — «Политика») описывает порядок обработки персональных данных пользователей Telegram-бота МедБот СПб (далее — «Бот»), предоставляемого самозанятым Павлович Даниил Дмитриевич, ИНН 780247939298 (далее — «Оператор»).

<b>1. Общие положения</b>

1.1. Политика разработана в соответствии с Федеральным законом РФ №152-ФЗ «О персональных данных».

1.2. Использование Бота означает согласие пользователя с настоящей Политикой.

<b>2. Состав собираемых данных</b>

2.1. Оператор собирает следующие данные:

- Telegram ID пользователя.
- ФИО, номер полиса ОМС (для записи к врачу).
- Email (для отправки чеков при оплате).
- Данные о записях к врачам (дата, время, врач, клиника).

2.2. Данные предоставляются пользователем добровольно при использовании Бота.

<b>3. Цели обработки данных</b>

3.1. Данные обрабатываются для:

- Предоставления услуг автоматической записи к врачам через портал Горздрав.
- Обработки платежей и отправки электронных чеков.
- Отправки уведомлений о статусе записи.
- Обеспечения технической поддержки.

<b>4. Защита данных</b>

4.1. Данные хранятся на защищенных серверах с использованием шифрования (HTTPS, SSL).

4.2. Доступ к данным имеют только уполномоченные сотрудники Оператора.

4.3. Данные не передаются третьим лицам, за исключением:

- Платежной системы ЮKassa для обработки платежей.
- Случаев, предусмотренных законодательством РФ.

<b>5. Срок хранения данных</b>

5.1. Данные хранятся в течение срока использования Бота и 6 месяцев после последней активности пользователя.

5.2. Пользователь может запросить удаление данных, обратившись в поддержку (@vispar_work).

<b>6. Права пользователя</b>

6.1. Пользователь имеет право:

- Запросить информацию о собранных данных.
- Отозвать согласие на обработку данных.
- Требовать удаления данных.

6.2. Для реализации прав пользователь может обратиться в поддержку (@vispar_work).

<b>7. Заключительные положения</b>

7.1. Оператор оставляет за собой право вносить изменения в Политику, уведомляя пользователей через Бот.

7.2. Контакты для связи: @vispar_work
